SYSTEM SPECIFICATION
Full CRM Platform for Tax Refund & Accounting Firm

You are a senior full-stack engineer.

Your task is to build a production-ready CRM platform for managing a tax refund and accounting firm.

This system replaces an existing CRM that was previously built in Base44.

The goal is to recreate the system with stronger architecture, backend automation, webhook capabilities, and long-term scalability.

The system must be production ready, modular, and designed for future SaaS expansion.

AUTOMATION ENGINE

Create a background job using cron.

Runs every hour.

Logic:

Fetch active TaskTemplates.

Fetch all active self_employed clients.

For each template and client:

Check if AutoTask exists for the current period.

If not:

Calculate due date.

If current date within advance_days window:

Create AutoTask

Create Task

Generate description using description_template with dynamic variables.

Assign task to

client.assigned_accountant_id

or fallback admin.

REMINDER SYSTEM

Run check every minute.

If reminder_datetime reached

trigger notification

If recurrence_pattern exists

generate next reminder
